My First Unity Game

My first Unity game was an interesting challenge for me because, well, it was my first time touching Unity. The gameplay involved players navigating through various obstacles to reach a portal. The development process was made easier by following a well-made YouTube tutorial by Jonas Tyroller (https://www.youtube.com/@JonasTyroller). The game features moving platforms, narrow paths, and treacherous gaps. Creating this game ignited my passion for game development, and I realized that with some time and effort, I could create some great games.
